If the region $\Omega$ is given by $a\le x\le b$, $\phi(x)\le y\le\phi_2(x)$ (This is called a Type 1 region), then $$\iint_\Omega f(x,y)dxdy=\int_a^b\int_{\phi_1(x)}^{\phi_2(x)}f(x,y)dxdy=\int_a^b[\int_{\phi_1(x)}^{\phi_2(x)}f(x,y)dy]dx\ne\int^{\phi_2(x)}_{\phi_1(x)}\int^b_af(x,y)dxdy$$
If the region $\Omega$ is given by $c\le y\le d, \psi_1(y)\le x\le\psi_2(y)$(this is called a Type II region), then $$\iint_\Omega f(x,y)dxdy=\int_c^d\int_{\psi_1(y)}^{\psi_2(y)}f(x,y)dxdy=\int_c^d[\int_{\psi_1(y)}^{\psi_2(y)}f(x,y)dx]dy\ne\int^{\psi_2(y)}_{\psi_1(y)}\int^d_cf(x,y)dydx$$Applications of double integrals include

Volume: $V=\iint_\Omega f(x,y)dxdy,\ f(x,y)>0,f(x,y)$ is top and $\Omega$ is base

Area: $A=\iint_\Omega dxdy,\Omega$ is region to find area of f(x,y)

## Examples

1. Evaluate $\iint_\Omega x^2y\ dxdy$  taking $\Omega:0\le x\le 1,0\le y\le x$ 
	- Type 1: $0\le x\le 1,\ 0\le y\le x$$$\iint_\Omega f(x,y)dydx=\int_0^1\int_0^xx^3y\ dydx=\int^1_0[x^3\frac{1}{2}y^2\vert^x_0]=\int_0^1\frac{x^5}{2}=\frac{1}{12}$$
	- Type 2: $0\le y\le 1,\ y\le x\le 1$ $$\iint_\Omega f(x,y)dxdy=\int_0^1\int_y^1x^3y\ dxdy=\int^1_0[\frac{1}{4}x^4\cdot y|^1_y]=[\frac{1}{4}\cdot\frac{1}{2}y^2-\frac{1}{4}\cdot\frac{1}{6}y^6]^1_0=\frac{1}{12}$$
2. Calculate by double integration of the area bound by the curves $y=x$ and $x=4y-y^2$
	- $x=4y-y^2$ is a parabola that opens to the left
	- $y=4y-y^2$
		- $y^2-3y=0$
		- $y=0,\ 3$
	- Points are $(0,0)\ (3,3)$
	- Type 2 description because if we did Type 1 we would have to break it up
	- $0\le y\ \le 3, y\le x\le 4y-y^2$
	- $$A=\iint_\Omega 1dxdy=\int^3_0\int_y^{4y-y^2}dxy=\int_0^3[4y-y^2-y]dy=\int_0^3(3y-y^2)dy=[\frac{3}{2}y^2-\frac{1}{3}y^3]^3_0=\frac{9}{2}$$
3. Give the formula for the volume under the paraboloid $z=x^2+y^2$ within the cylinder $x^2+y^2\le 1,\ z\ge0$
	 - Type 1:
		 - $-1\le x\le 1, -\sqrt{1-x^2}\le y\le \sqrt{1-x^2}$
	- z-values: $0\le z\le x^2+y^2=f(x,y)$
	- $$Vol=\int^1_{-1}\int^{\sqrt{1-x^2}}_{-\sqrt{1-x^2}}\int^{x^2+y^2}_01dzdydx$$
4. Calculate $\int_0^2\int_{x^2}^42xcos(y^2)dydx$ by changing the order of integration **Extremely Important!!**
	- Type 1: $0\le x\le 2,\ x^2\le y\le 4$
	- Why can't we solve this way?
		- There is NO anti derivative for $cos(y^2)$
	- Type 2: $0\le y\le 4$, $0\le x\le\sqrt{y}$ $$\int_0^2\int_{x^2}^42xcos(y^2)dxdy=\int_0^4x^2cos(y^2)|^\sqrt{y}_0dy=\int_0^4ycos(y^2)dy=\frac{1}{2}sin(y^2)|^4_0=\frac{1}{2}sin(16)$$
5. Find the volume of the solid bounded by the coordinate planes and the plane $\frac{x}{2}+\frac{y}{3}+\frac{z}{4}=1$
	- Intercepts $(2,0,0), (0,3,0), (0,0,4)$
	- This solid is a Tetraherdron
	- solve for $z=4(1-\frac{1}{2}x-\frac{1}{3}y)$ 
	- $f(x,y)=4(1-\frac{1}{2}x-\frac{1}{3}y)$
	- What is $\Omega$? The triangle on the $xy$ coordinate plane ($z=0$)
	- $\implies \frac{1}{2}x+\frac{1}{3}y=1$ or $y=3(1-\frac{1}{2}x)$
	- Type 1: $0\le x\le 2$, $0\le y\le 3(1-\frac{1}{2}x)$$$Vol =\int_0^2\int_0^{3(1-\frac{1}{2}x)}4(1-\frac{1}{2}x-\frac{1}{3}y)dydx$$
	- Pro Tip: On a tetrahedron, use u-subs
		- $u=1-\frac{1}{2}x-\frac{1}{3}y$, $du=-\frac{1}{3}dy\implies -3du=dy$
		- $t=1-\frac{1}{2}x\implies dt=-\frac{1}{2}dx\implies dx=-2dt$$$=4\int_0^3(-3)(\frac{1}{2})(1-\frac{1}{2}x-\frac{1}{3}y)^2|_0^{3(1-\frac{1}{2}x)}dx=4\int_0^4+\frac{3}{2}[(1-\frac{1}{2}x-\frac{1}{3}3(1-\frac{1}{2}x))]dx=6\int_0^2(1-\frac{1}{2}x)^2dx=6\cdot(-2)(\frac{1}{3})(1-\frac{1}{2}x)^3|^2_0=4[(1-0)^3-(1-\frac{1}{2}\cdot2)^3]=4$$
