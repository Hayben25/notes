
|                    | Retangular                                                                                     | Polar                             |
| ------------------ | ---------------------------------------------------------------------------------------------- | --------------------------------- |
| **Volume**         | $\iint_\Omega f(x,y)dxdy$                                                                      | $\iint_\tau F(r,\theta)drd\theta$ |
| **Area**           | $\iint 1dxdy$                                                                                  | $\iint_\tau rdrd\theta$           |
| **Mass**           | $M=\iint_\Omega\lambda(x,y)dxdy$, $\lambda =density\ function$                                 |                                   |
| **Center of Mass** | $x_M=\frac{\iint_\Omega \lambda(x,y)xdxdy}{M}$, $y_M=\frac{\iint_\Omega \lambda(x,y)ydxdy}{M}$ |                                   |
## Examples

1. Find the mass and center of mass where $\Omega$ is the triangle with verities $(0,0)$, $(1,3)$, and $(1,5)$. $\lambda (x,y)=xy$
	- 3 lines
		- $y=3x$, $y=5x$, $x=1$
	- $\Omega$ as Type 1: $0\le x\le 1$, $3x\le y\le 5x$
$$M=\iint_\Omega \lambda(x,y)dxdy=\int_0^1\int_{3x}^{5x}xydydx=\dots=2$$
$$x_M=\frac{1}{2}\int_0^1\int_{3x}^{5x}xy\cdot xdydx=\dots=\frac{4}{5}$$
$$y_M=\frac{1}{2}\int_0^1\int_{3x}^{5x}xy\cdot ydyx=\dots=\frac{49}{15}$$
