In section we found absolute extrema of a function on a region that contained its boundary (like we did in Calc 1 when we used a closed and bounded interval). Now we will look at a different process - we will be optimizing a function *f* subject to a constraint $g=0$. Let *f* be a function to be optimized

Method of Lagrange(stupid fr\*nch):
- ($\lambda$ is called the *Lagrange multiplier*)
- Set your constraint function = 0 (that's g)
- Then solve the system $\nabla f(x,y,z)=\lambda\nabla g(x,y,z)$ and $g(x,y,z)=0$ 
- Last, plug in all solutions (x,y,z) into *f* to identify max and min values (if they exist)

## Examples

1. Maximize $xy$ on the ellipse $4x^2+9y^2=36$
	- Let $f(x,y)=xy$, $g(x,y)=4x^2+9y^2-36$, $g=0$
	- $\nabla f(x,y)=(y,x)$, $\nabla g(x,y)=(8x,18y)$
	- Set $\nabla f=\lambda \nabla g$:
		- $y=\lambda 8x$
		- $x=\lambda 18y$
		- $\implies \frac{y}{8x}=\frac{x}{18y}=\lambda$ 
		- Writing this assumes $x | y \ne 0$
			- Note if $x|y=0$, $f(x,y)=xy=0$ but that's not the max
		- $18y^2=8x^2\implies 9y^2=4x^2$
	- $g=0$: $36=4x^2+9y^2=8x^2\implies\frac{36}{8}=\frac{9}{2}\implies x=\pm\frac{3}{\sqrt{2}}$
	- $\implies y=\pm\sqrt{2}$
	- How many points? Since $4x^2=9y^2$ allows $x$ and $y$ to share or differ in sign, we get 4 points
	- Check all 4 possible points

| Point                             | f(x) |
| --------------------------------- | ---- |
| $(\frac{3}{\sqrt{2}},\sqrt{2})$   | 3    |
| $(\frac{3}{\sqrt{2}},-\sqrt{2})$  | -3   |
| $(-\frac{3}{\sqrt{2}},\sqrt{2})$  | -3   |
| $(-\frac{3}{\sqrt{2}},-\sqrt{2})$ | 3    |

1. Minimize $x+2y+4z$ on the sphere $x^2+y^2+z^2=7$
	- Let $f(x,y)=x+2y+4z$, $g(x,y,z)=x^2+y^2+z^2-7$, $g=0$
	- $\nabla f(x,y,z)=(1,2,4)$, $\nabla g(x,y,z)=(2x,2y,2z)$
	- $\nabla g=\lambda \nabla f$:
		- $2x=\lambda$
		- $2y=2\lambda$
		- $2z=4\lambda$
		- $\implies \lambda =2x=y=\frac{z}{2}$
			- $2x=y$
			- $\frac{z}{2}=y\implies z=2y=4x$
			- $y=2x$, $z=4x$
	- $g=0$:
		- $7=x^2+y^2+z^2=x^2+(2x)^2+(4x)^2=21x^2$
		- $\implies x=\pm\frac{1}{\sqrt{3}}$, $y=2x=\pm\frac{2}{\sqrt{3}}$, $z=4x=\pm\frac{4}{\sqrt{3}}$
	- How many points? Since $y$ and $z$ vary with $x$ there are only 2

| Point                                                           | f(x)         |
| --------------------------------------------------------------- | ------------ |
| $(\frac{1}{\sqrt{3}},\frac{2}{\sqrt{3}},\frac{4}{\sqrt{3}})$    | $7\sqrt{3}$  |
| $(-\frac{1}{\sqrt{3}},-\frac{2}{\sqrt{3}},-\frac{4}{\sqrt{3}})$ | $-7\sqrt{3}$ |
