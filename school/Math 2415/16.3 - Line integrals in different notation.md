- Given a vector function $h(x,y)=P(x,y)\hat{i}+Q(x,y)\hat{j}$ and a curve $C:r(u)=x(u)\hat{i}+y(u)\hat{j},\ a\le u\le b$ then$$\int_Ch(r)\cdot dr=\int_CP(x,y)dx+Q(x,y)dy$$
	- Why? $\overrightarrow{r}=(x,y)\implies d\overrightarrow{r}=(dx,dy)$
		- $\overrightarrow{h}(\overrightarrow{r})=\overrightarrow{h}(x,y)=P(x,y)\hat{i}+Q(x,y)\hat{j}$
		- Thus $\overrightarrow{h}(\overrightarrow{r})\cdot d\overrightarrow{r}=(P(x,y),Q(x,y))\cdot(dx,dy)$
- Green's Theorem
	- We said earlier that a curve $C$: $\overrightarrow{r}(u)=x(u)\hat{i}+y(u)\hat{j},\ a\le u\le b$ is a closed curve if $\overrightarrow{r}(a)=\overrightarrow{r}(b)$
	- Well, C is a **simple closed curve** if it is closed and $\overrightarrow{r}(u_1)\ne\overrightarrow{r}(u_2)$ for all $u_1,\ u_2\in(a,b),\ u_1\ne u_2$
	- A simple closed curve is called a **Jordan curve** and the positive direction is counterclockwise. The region enclosed by a Jordan curve is a **Jordan region**
	- Green's Theorem: Given $\overrightarrow{h}(x,y)=P(x,y)\hat{i}+Q(x,y)\hat{j}$ and $C$, a simple closed curve oriented in the conterclockwise direction which encloses the Jordan region $\Omega$, then$$\oint_C\overrightarrow{h}(\overrightarrow{r})\cdot d\overrightarrow{r}=\oint_C(P(x,y)dx+Q(x,y)dy)=\iint_\Omega(\frac{\partial Q}{\partial x}-\frac{\partial P}{\partial y})$$
- Finding area of $\Omega$ using Green's Theorem: If $C$ is a simple closed curve enclosing the region $\Omega$, then
	- Area of $\Omega=$$$\iint_\Omega1dxdy=\oint_C-ydx=\oint xdy=\frac{1}{2}\oint_C-ydx+xdy$$
		- Let $P=-y,\ Q=0$, $Pdy+Qdy=-ydx_0dy=-ydx$. So $$\oint_C-ydx=\oint_CPdx+Qdy=\iint_\Omega[0-(-1)]dxdy=\iint_\Omega1dxdy=Area$$
		- Let $P=0,\ Q=x$
			- $Q_x-P_y=1$

## Example

1. Find $\int_Ch(r)dr$ given $h(x,y)=x^2y\hat{i}+2xy\hat{j}$ and $r(u)=u^2\hat{i}+(1-u^3)\hat{j},\ 0\le u\le2$
	- $P(x,y)=x^2y,\ Q(x,y)=2xy$
	- Want $\int_CP(x,y)dx+Q(x,y)dy,\ x=u^2,\ y=1-u^3$
		- $dx=2udu,\ dy=-3u^2du$
	- $P(x,y)dx+Q(x,y)dy=(u^2)^2(1-u^3)\cdot 2udu+2(u^2)(1-u^3)(-3u^2du)=[2u^5-2u^8-6u^4+6u^7]du$
	- $\int_Cx^2ydx+2xydy=\int_0^2[2u^5-2u^8-6u^4+6u^7]du=[\frac{1}{3}u^6-\frac{2}{9}u^9-\frac{6}{5}u^5+\frac{3}{4}u^8]^2_0=\frac{2752}{45}$
2. Evaluate $\int_C(x-2y)dx+2xdy$ along the parabolic path $C:y=2x^2$ from $(0,0)$ to $(1,2)$
	- $\overrightarrow{h}(x,y)=(x-2y,2x)=(P(x,y),Q(x,y))$
	- Parameterize: Let $x=u,\ y=2u^2,\ u\in[0,1]$ 
		- $P=x-2y=u-4u^2,\ Q=2u$
		- $dx=du,\ dy=4udu$
	- Now, $$\int_CPdy+Qdy=\int_0^1[(u-4u^2)du+(2u)(4udu)]=\int_0^1[u-4u^2+8u^2]du$$
3. Calculate $\oint_Cx^2ydx+2xydy$ where $C$ is the square with vertices $(0,0)$, $(1,0)$, $(1,1)$ and $(0,1)$ traversed counterclockwise
	- $P=x^2y,\ Q=2xy$
	- $P_y=x^2,\ Q_x=2y$
	- $Q_x-P_y=2y-x^2$
	- $\Omega:\ 0\le x\le1,\ 0\le y\le 1$ $$\int_0^1\int_0^1[2y-x^2]dydx=\int_0^1[y^2-x^2y]^1_0dx=\int_0^1[1-x^2]dx=[x-\frac{1}{3}x^3]^1_0=1-\frac{1}{3}=\frac{2}{3}$$
4. Calculate $\oint_C(e^{sinx}-y^3)dx+(x^3+\sqrt{y^4+1})dy$ where $C$ is the circle $x^2+y^2=4$
	- $P_y=-3y^2,\ Q_x=3x^2$
	- $Q_x-P_y=3x^2+3y^2$
	- $\oint_CPdx+Qdy=\iint_\Omega(3x^2+3y^2)dxdy$
		- $\Omega:0\le\theta\le2\pi,\ 0\le r\le2$$$\int_0^{2\pi}\int_0^23r^2\cdot rdrd\theta=2\pi\int_0^33r^2dr=2\pi\cdot\frac{3}{4}r^4|^3_0=24\pi$$
5. Calculate $\oint_C(3x^2y^2+2x)dx+(2x^3y-4y-1)dy$ where $C$ is the simple closed curve formed by the upper semi-circle of radius 2 and the line segment connecting (-2,0) to (2,0)
	- $P_y=6x^2y,\ Q_x=6x^2y$
	- $Q_x-P_y=6x^2y-6x^2y=0$
	- $\oint_CPdx+Qdy=\iint_\Omega0dxdy=0$ 
6. Derive the formula for the area enclosed by the ellipse $\frac{x^2}{a^2}+\frac{y^2}{b^2}=1$ using Green's theorem
	- Parameterize: $(\frac{x}{a})^2+(\frac{y}{b})^2=1$
		- Let $x=acos(u),\ y=bsin(u),\ u\in[0,2\pi]$
			- $dx=-asin(u)du,\ dy=bcos(u)du$
	- Use $Area=\frac{1}{2}\int_C-ydx+xdy$ 
	- $-ydx=(-bsin(u))(-asin(u)du)=absin^2(u)du$
	- $xdy=acos(u)bcos(u)du=avcos^2(u)du$
	- $-ydx+xdy=abdu$$$Area=\iint_\Omega dxdy$$
