- Given $h(x,y,z)=h_1(x,y,z)\hat{i}+h_2(x,y,z)\hat{j}+h_3(x,y,z)\hat{k}$, a vector function (vector field) defined on some region $\Omega$ in space, and a smooth curve (or piecewise smooth curve)$$C:\ r(u)=x(u)\hat{i}+y(u)\hat{j}+z(u)\hat{k}$$
- Or given $h(x,y)=h_1(x,y)\hat{i}+h_2(x,y)\hat{j}$, a vector function defined on some region $\Omega$ in the plane, and a smooth curve (or piecewise smooth curve)$$C:r(u)=x(u)\hat{i}+y(u)\hat{j},\ a\le u\le b\ in\ \Omega$$
- The **line integral** of h over the curve $C$, denoted by $\int_C h(r)\cdot dr$ is the the number given by: $$\int_Ch(r)\cdot dr=\int_a^bh(r(u))\cdot r'(u)du$$
- Properties of line integrals
	1. $\int_Ch(r)\cdot dr$ is invariant under order preserving prameterizations of C
	2. Reversing the direction of C changes the sign of $\int_Ch(r)\cdot dr$
		- For example using #2 and $C_3$: $(3-2u)\hat{i}+(9-8u)\hat{j},\ 0\le u\le 1$
	3. If a curve $C$ is made up of connected smooth pieces $C_1,\ C_2,\ C_3$ then
		- $\int_Ch(r)\cdot dr=\int_{C_1}h(r)\cdot dr+\int_{C_2} h(r)\cdot dr+\dots+\int_{C_n} h(r)\cdot dr$ 
- Line integral of a Scalar Field
	- use arc length
	- $\int_Cf(x,y)ds=\int_a^bf(x(u),y(u))\sqrt{x'(u)^2+y'(u)^2}du$ 

## Examples

1. Set $h(x,y,z)=xy\hat{i}+yz\hat{j}+xz\hat{k}$ and let $C$ be the curve $r(u)=u\hat{i}+u^2\hat{j}+u^3\hat{k},\ 0\le u\le 2$. Find $\int_C h(r)\cdot dr$
	- $\overrightarrow{h}(\overrightarrow{r})=\overrightarrow{h}(\overrightarrow{r}(u))=\overrightarrow{h}(u,u^2,u^3)=u^3\hat{i}+u^5\hat{k}+u^4\hat{k}$
	- $d\overrightarrow{r}=\overrightarrow{r}(u)du=(1,2u,3,u^2)du$
	- $\overrightarrow{h}(\overrightarrow{r})\cdot\overrightarrow{r}$
	- $\frac{668}{7}$
2. Set $h(x,y)=(xy+2y)\hat{i}+(2x+y)\hat{j}$, find $\int_Ch(r)]\cdot dr$ on:
	- $C_1$: The parabola $y=x^2$ from (1,1) to (3,9)
		- $\overrightarrow{r}(u)=(x(u),y(u)),\ u\in[a,b]$
			- $x=u,\ y=u^2,\ u\in[1,3]$
			- $\overrightarrow{r}(u)=(u,u^2),\ u\in[1,3]$
		- $\overrightarrow{h}(\overrightarrow{r}(u))=\overrightarrow{h}(u,u^2)=(u^3+2u^2, 2u+u^2)$ 
			- $r'(u)=(1,2u)$
		- $\overrightarrow{h}(\overrightarrow{r}(u))=(u^3+2u^2,2u+u^2)\cdot(1,2u)=3u^7+6u^2$
		- Now $$\int_{C_1}\overrightarrow{h}(\overrightarrow{r})\cdot dr=\int^3_1(3u^7+6u^2)$$
	- $C_2$: The line segment from (1,1) to (3,9)
		- $\overrightarrow{r}(u)=(1,1)+u(3-1,9-1)=(1,1)+u(2,8)=(1+2u,1+8u),\ u\in[0,1]$
		- $\overrightarrow{h}(\overrightarrow{r}(u))=h(1+2u,1+8u)=(3+26u+16u^2,3+12u)$
			- $\overrightarrow{r}'(u)=(2,8)$
		- $\overrightarrow{h}(\overrightarrow{r}(u))\cdot \overrightarrow{r}'(u)=30+48u+32u^2$
		- $$\int_{C_2}\overrightarrow{h}(\overrightarrow{r}(u)\cdot d\overrightarrow{r}=\int_1^3(3u^3+6u^2)du=[\frac{3}{4}u^4+2u^3]=\frac{344}{3}$$
3. Set $h(x,y,z)=xy\hat{i}+x^2z\hat{j}+xyz\hat{k}$ and calculate $\int_Ch(r)\cdot dr$ over the path
	- $C$: The line segment from (0,0,0) to (1,1,1)
		- $\overrightarrow{r}(u)=(0,0,0)+u(1,1,1)=(u,u,u)$
		- $\overrightarrow{h}(\overrightarrow{r}(u))\cdot \overrightarrow{r}'(u)=(u^2+u^3+u^3)\cdot (1,1,1)=u^2+2u^3$
		- $\int_0^1(u^2+2u^3)du$
	- $C$: The parabolic segment $y=x^2$ from (0,0,0) to (1,1,0) followed by the line segment from (1,1,0) to (1,1,1)
		- $C_1:\ (0,0,0)\rightarrow(1,1,0)$ along $y=x^2$
			- $\overrightarrow{r_1}(u)=(u,u^2,0)\ u\in[0,1]$
		- $C_2:\ (1,1,1) along line$
			- $\overrightarrow{r_2}(u)=(1,1,u),\ u\in[0,1]$ 
		- $\overrightarrow{h}(x,y,z)=(xy,x^2z,xyz)$
		- $C_1:\ \overrightarrow{h}(\overrightarrow{r_1}(u))=\overrightarrow{h}(u,u^2,0)=(u^3,0,0),\ \overrightarrow{r_1}'(u)=(1,24,0)$
			- $\overrightarrow{h}(\overrightarrow{r_1}(u))\cdot \overrightarrow{r'}(u)=u^3$
	- $C:r(u)=(1-u)\hat{i}+(1-u)\hat{j}+(1-u)\hat{k}$, $0\le u\le 1$
		- C is line segment from (1,1,1)
		- $-\frac{5}{6}$
4. Evaluate the line integral over the function $f(x,y)=xy$ over the unit circle
	- Want $\int_C xy\cdot ds$ where C is unit circle
	- $C:\overrightarrow{r}(u)=(cos(u),sin(u)),\ u\in[0,2\pi]$
	- $f(\overrightarrow{r}(u))=f(cos(u),sin(u))=cos(u)sin(u)$
		- $x'(u)=-sin(u)$
		- $y'(u)=cos(u)$
	- $ds=\sqrt{x'(u)^2+y'(u)^2}=\sqrt{sin^2(u)+cos^2(u)}=1\ du=du$
	- Thus, $\int_Cxy\ ds=\int_0^{2\pi} cos(u)sin(u)du=\frac{1}{2}sin^2(u)|_0^{2\pi}=0$
