- What was the MVT for functions of one variable?
	- if *f*(x) is continous on \[a,b] and differentiable on (a,b) then there is at least one c$\in (a,b)$ where $f'(c)=\frac{f(b)-f(a)}{b-a}$
- Theorem 1. (The Mean Value Theorem for Functions of Several Values). If *f*, a function of several variables, is differentiable at each point for the line segment $\overline{ab}$ then there exists on that line segment a point c between b such that
	- $f(b)-f(a)=\nabla f(c)\cdot (b-a)$
	- What is $\overline{ab}$?
		- $r'(0)=\overrightarrow{a}+0(\overrightarrow{b}-\overrightarrow{a})=\overrightarrow{a}$
		- $r'(1)=\overrightarrow{a}+1(\overrightarrow{b}-\overrightarrow{a})=\overrightarrow{b}$
	- A nonempty set *U* (in the plane or 3-space) is said to be connected if any 2 points of *U* can be joined by a polygonal path that lies entirely in *U*
		- Theorem - Let *U* be an open connected set and let *f* be a differentiable function on *U*. If $\nabla f(x)=0$ for all x in *U*, then *f* is constant on *U*
		- Theorem - Let *U* be an open connected set and let *f* and *g* be differentiable functions on *U*. If $\nabla f(x)=\nabla g(x)$ for all x in *U*, then *f* and *g* differ by a constant on *U* 
- Theorem 2. (Chain Rule Along a Curve) If *f* is continuously differentiable on an open set *U* and $r=r(t)$ is a differentiable cure that lies in *U*, then the composition $f \circ r$ is differentiable and 
	- $\frac{d}{dt}[f(r(t))]=\nabla f(r(t))\cdot r'(t)$
- Other versions of the chain rule:
	- If
		- $u=u(x,y)$ where $x=x(s,t)$ and $y=y(s,t)$
	- Then
		- $\frac{\partial u}{\partial s}=\frac{\partial u}{\partial x}\frac{\partial x}{\partial s}+\frac{\partial u}{\partial y}\frac{\partial y}{\partial s}$ and $\frac{\partial u}{\partial t}=\frac{\partial u}{\partial x}\frac{\partial x}{\partial t}+\frac{\partial u}{\partial y}\frac{\partial y}{\partial t}$
	- And if
		- $u=u(x,y,z)$ where $x=x(s,t)$ and $y=y(s,t)$, and $z=z(s,t)$
	- Then
		- $\frac{\partial u}{\partial s}=\frac{\partial u}{\partial x}\frac{\partial x}{\partial s}+\frac{\partial u}{\partial y}\frac{\partial y}{\partial s}+ \frac{\partial u}{\partial y}\frac{\partial z}{\partial s}$ and $\frac{\partial u}{\partial t}=\frac{\partial u}{\partial x}\frac{\partial x}{\partial t}+\frac{\partial u}{\partial y}\frac{\partial y}{\partial t}+\frac{\partial u}{\partial y}\frac{\partial z}{\partial s}$

	- $f(b)-f(a)=\nabla f(c)\cdot (b-a)$
	- $\nabla f(x,y)=(3x^2-y,-x)$
	- $f(\overrightarrow{b})=f(1,3)=0$
	- $\overrightarrow{b}-\overrightarrow{a}=(1,3)-(0,1)=(1,2)$
	- What is $\overline{ab}$?
		- $\overrightarrow{r}(t)=(0,1)+t(1,2)=(t,1+2t)$
	- What is $\overrightarrow{c}$? some point on this segment between $\overrightarrow{a}$ and $\overrightarrow{b}$, meaning $\overrightarrow{r}(t)$ for some $t\in (0,1)$
		- $\nabla f(\overrightarrow{c})=\nabla f (\overrightarrow{c})\cdot(\overrightarrow{b}-\overrightarrow{a})$ 
		- $-2=3t^2-1-2t-2t$
		- $0=3t^2-4t+1$
		- $0=(3t-1)(t-1)$
		- $t=\frac{1}{3},1$
		- Actually only $t=\frac{1}{3}$ because 1 is not $t\in (0,1)$
