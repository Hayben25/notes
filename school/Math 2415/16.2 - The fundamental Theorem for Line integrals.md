- Given a vector function h and a curve $C: r(u),\ a\le u\le b$. If h is the gradient of function $f$, i.e $\nabla f=h$ then
	- $\int \nabla f(r)\cdot dr=f(r(b))-f(r(a))$
	- (Remember the vector function $h(x,y)=P(x,y)\hat{i}+Q(x,y)\hat{j}$ is the gradient of $f\iff\frac{\partial P}{\partial y}=\frac{\partial Q}{\partial x}$)
	- Recall: Fund. Theorem of Calculus
		- $\int_a^bf'(x)dx=f(b)-f(a)$ 
- Corollary
	- The curve $C: r(u)=x(u)\hat{i}+y(u)\hat{j},\ a\le u\le b$ is a closed curve is $r(a)=r(b)$
	- Corollary 1: If $C:r(u)$ is a closed curve then $\int_C\nabla f(r)\cdot dr=0$
	- Corollary 2: If $C_1$ and $C_2$ are any 2 curves that begin with A=r(a) and end with B=r(b) then $\int_{C_1}\nabla f(r)\cdot dr=\int_{C_2}\nabla f(r)\cdot dr$
## Example

1. Find $\int_Ch(r)\cdot dr$ given $h(x,y)=(3x^2y^3+2x+1)\hat{i}+(3x^3y^2-4y+\pi cos(\pi y))\hat{j}$ and $C$ is the curve traced out by $r(u)=(u^3-3u)\hat{i}+(2-u^2)\hat{j},\ 0\le u\le2$
	- Is $\overrightarrow{h}=\nabla f$ (yes b/c $P_y=Q_x$)
	- Find $f:\ \int Pdx=\int(3x^2y^3+2x+1)dx=x^3y^3+x^2+x+\phi(y)$
		- Set $f(x,y)=x^3y^3+x^2+x+\phi(y)$
		- $f_y=3x^3y^2+\phi'(y),\ Q=3x^3y^2-4y+\pi cos(\pi y)\implies \phi(y)=-2y^2+sin(\pi y)+C$
	- I use $f(x,y)=x^3y^3+x^2+x-2y^2+sin(\pi y)$
		- $\overrightarrow{r}(0)=(0,2),\overrightarrow{r}=(2,-2)$
	- $\int_C\overrightarrow{h}(\overrightarrow{r})\cdot dr=f(2,-2)-f(0,2)=-2^6+2^2+2-2^3+2^3=-58$
