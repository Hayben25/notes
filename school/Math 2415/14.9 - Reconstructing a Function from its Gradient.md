The gradient of a function $f(x,y)=\frac{\partial f}{\partial x}\hat{i}+\frac{\partial f}{\partial y}\hat{j}$

So if we have a (gradient) vector in the form
- $P(x,y)\hat{i}+Q(x,y)\hat{j}$
- We need
	- $P(x,y)=\frac{\partial f}{\partial x}(x,y)=f_x(x,y)$
	- $Q(x,y)=\frac{\partial f}{\partial y}(x,y)=f_y(x,y)$

Now when integrating $\frac{\partial f}{\partial x}$ with respect to $x$, remember that $y$ is a constant 

## Example

1. $xy^2\hat{i}+x^2y\hat{j}$
	- $\int (xy^2)dx=\frac{1}{2}x^2y^2+\phi (y)$
	- Set $f(x,y)=\frac{1}{2}x^2y^2+\phi(y)$ 
		- $f_y(x,y)=x^2y+\phi '(y)$
		- Want $f_y(x,y)=x^2+y$:
			- $x^2y+\phi '(y)=x^2y\implies \phi '(y)=0\implies \phi (y)=C$
	- Thus $f(x,y)=\frac{1}{2}x^2y^2+C$
	- Note: Sometimes the given vector could not be the gradient of a function. Can you think of a quick way to determine if a vector is a gradient? no
		- If $P(x,y)\hat{i}+Q(x,y)\hat{j}=\nabla f\implies P_y=Q_x=f_{xy}=f_{yx}$
2. $(y^2+x)\hat{i}+(x^2+y)\hat{j}$
	- $P_y=3y^2\ne Q_x=2x$
3. $(e^x+2xy)\hat{i}+(x^2+sin\ y)\hat{j}$
	- $P_y=Q_x=2x$
	- $\int Pdx=\int (e^x+2xy)dx=e^x+x^2y+\phi (y)$
	- Set $f(x,y)e^x+x^2y+\phi (y)$
		- $f_y=x^2+\phi '(y)$, $Q=x^2+sin(y)$
			- $\implies $\phi (y)=-cos(y)+C$
	- Thus, $f(x,y)=e^x+x^2y-cos(y)+C$
4. $(y^2+x)\hat{i}+(x^2+y^2)\hat{j}=P\hat{i}+Q\hat{j}$
	- $P_y=3y^2\ne Q_x=2x$
	- Thus the example cannot be a gradient
