So far, we have used Polar, Cylindrical, and Spherical coordinates to make some of our integration problems easier. In calc 1, we used u-sub to make some integration easier. We are now going to do something similar but now we will change the xy-coordinates into uv-coordinates. To do that, we will need to transform our region too.

- To change our variables in a double integral using our transformation, we first need to find the **Jacobian**
- The Jacobian of the transformation $(x,y)\rightarrow(x(u,v),y(u,v))$ is
	- Full story: The Jacobian is really the matrix but we only need the determinant$$J(u,v)=|^\frac{\partial x}{\partial u}_\frac{\partial x}{\partial v}\ ^\frac{\partial y}{\partial u}_\frac{\partial y}{\partial v}|=\frac{\partial x}{\partial u}\cdot\frac{\partial y}{\partial v}+\frac{\partial y}{\partial u}\cdot\frac{\partial x}{\partial v}$$
- Now if the Jacobian isn't 0 then the areao f $\Omega-\iint_\Gamma|J(u,v)|dudv$ where $\Gamma$ represents
- And so $\iint_\Omega f(x,y)dxdy=\iint_\Gamma f(x(u,v),y(u,v))|J(u,v)|dudv$
	- $dxdy=|J(u,v)|dudv$
- When changing variable in a tirple integral we make trhee coordinate changes$$x=x(u,v,w),\ y=y(u,v,w),\ z=z(u,v,w)$$
- If these functions carry a basic solid $\Gamma$ onto a solid $T$, then under conditions analogous to the 2D case, $$volume\ of\ T=\iiint |J(u,v,w)|dudvdw$$
## Example

1. Transform the given region $\Omega:\ x^2+\frac{y^2}{36}=1$ into $\tau$ using the transformation $x=\frac{1}{2}u,\ y=3v$
	- $1=x^2+\frac{y^2}{36}=(\frac{1}{2}u)^2+\frac{1}{36}(3v)^2=\frac{1}{4}u^2+\frac{9}{36}v^2=\frac{1}{4}(u^2+v^2)\implies 4=u^2+v^2$
2. Find the volume of the solid bounded above by the cone $z^2=x^2+y^2$, below by the xy-plane, and on the sides by the cylinder $x^2+y^2=2x$
	- z-limits: $0\le z\le\sqrt{x^2+y^2}=\sqrt{r^2}=r$
	- $x^2+y^2=2x$
		- $r^2=2rcos\theta$
		- $r=2cos\theta$
		- Polar:
			- $-\frac{\pi}{2}\le\theta\le\frac{\pi}{2}$
			- $0\le r\le 2cos\theta$
	- $$\int_{-\frac{\pi}{2}}^\frac{\pi}{2}\int_0^{2cos\theta}\int_0^rr\ dzdrd\theta=\int_{-\frac{\pi}{2}}^\frac{\pi}{2}\int_0^{2cos\theta}r^2\ drd\theta=\frac{1}{3}\int_{-\frac{\pi}{2}}^\frac{\pi}{2}8\cdot cos^3\theta\ d\theta=\frac{8}{3}\int_{-\frac{\pi}{2}}^\frac{\pi}{2}(1-sin^2\theta)cos\theta\ d\theta=\frac{8}{3}[sin\theta-\frac{1}{3}sin^2\theta]^\frac{\pi}{2}_{-\frac{\pi}{2}}$$
3. Set up the integral in spherical coordinates for the volume of the solid inside of $x^2+y^2+z^2=4$ and outside of $x^2+y^2=3z^2$, with $x\ge 0$ and $y\le 0$
	- $x^2+y^2+z^2=4\implies\rho^2=4\implies\rho=2$
	- $x^2+y^2=3z^2\implies r^2=3z^2\implies r=\sqrt3|z|$
	- $r=\sqrt3z\implies\rho sin\phi=\sqrt3\rho cos\phi\implies\phi=\frac{\pi}{3}$
	- $r=\sqrt3z\implies\rho sin\phi=-\sqrt3\rho cos\phi\implies\phi=\frac{2\pi}{3}$
	- Spherical: $\frac{\pi}{3}\le\phi\le\frac{2\pi}{3}$, $0\le\rho\le2$
		- $\theta$? $x\gt 0$, $y\le 0\implies \frac{3\pi}{2}\le\theta\le2\pi$
	- $$Vol=\int_\frac{3\pi}{2}^{2\pi}\int_\frac{\pi}{3}^{\frac{2\pi}{3}}\int_0^2\rho^2sin\phi\ d\rho d\phi d\theta$$
4. Find the Jacobian of the transformation $x=4uv$, $y=u^2+2v^2$

| $x=4uv$  | $y=u^2+2v^2$ |
| -------- | ------------ |
| $x_u=4v$ | $y_u=2u$     |
| $x_v=4u$ | $y_v=4v$     |
5. Evaluate $\iint_\Omega (x+y)cos(\pi(x-y))dxdy$ where $\Omega$ is the parallelogram bounded by $x-y=0$, $x-y=3$, $x+y=0$, $x+y=3$
	- 1$^{st}$ draw $\Omega$
	- Now decide on a transformation
		- Let $u=x-y$ and $v=x+y$
		- $\Gamma:\ 0\le u\le3,\ 0\le v\le3$
	- Write $x,y$ in terms of $u,v$:
		- $x-y=u$
		- $x+y=v$
		- $x=\frac{1}{2}(u+v)$, $y=\frac{1}{2}(v-u)$

| $x=\frac{1}{2}(u+v)$ | $y=\frac{1}{2}(v-u)$ |
| -------------------- | -------------------- |
| $x_u=\frac{1}{2}$    | $y_u=-\frac{1}{2}$   |
| $x_v=\frac{1}{2}$    | $y_v=\frac{1}{2}$    |
$J=\frac{1}{2}\cdot\frac{1}{2}-\frac{1}{2}\cdot(-\frac{1}{2})=\frac{1}{2}$ 
$$\iint_\Omega(x+y)cos(\pi(x-y))dxdy=\iint_\Gamma vcos(\pi u)|J|dudv=\int_0^3\int_0^3\frac{1}{2}vcos(\pi u)dvdu\frac{1}{2}\cdot\frac{1}{\pi}sin(\pi u)|^3_0v^2|^3_0=0$$
